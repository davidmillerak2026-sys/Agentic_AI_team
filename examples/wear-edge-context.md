# WearEdge Pro Context

WearEdge Pro is a wearable edge industrial multimodal AI Agent system.

Key assumptions:

- local-first and offline-capable deployment
- AR glasses for visual overlay
- bone-conduction headset for open-ear audio
- edge compute brain or Android phone as compute hub
- target scenarios include AR IQC, blind troubleshooting, analog OCR, and safety awareness
- downstream actions require stable schema contracts and human approval gates
